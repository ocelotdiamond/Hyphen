# Hyphen

Hyphen is a fairly fast remote desktop application designed to pilot a virtual machine. It provides a low-latency video connection to a virtual machine, and can restart the virtual machine automatically following a crash. The codebase was written in about 5 days and is held together with duct tape (which I ran out of halfway through), spit, and hopes and dreams.

## Requirements

Hyphen requires VirtualBox and Git Bash to be installed to run.
